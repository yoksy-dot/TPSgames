お一人様TPS 取説
このゲームは2017年3月の春休み期間に作成・ｗ・
ネット対戦まで実装したかったけどパソコン修理のためいったん完成としました

＜操作方法＞
移動 : WASDキーまたは矢印キー
ジャンプ : スペースキー
視点操作 : マウス
ズーム : マウスホイール
攻撃1 : 左クリック
攻撃2 : 右クリック
リロード : Rキー

＜キャラ紹介＞
*ボウガン兵
・攻撃１
割と普通 消滅までの時間は長め
・攻撃2
エフェクトの後、一定時間後にたくさんの攻撃が降ってくる
キューブには当たりにくい

*魔法使い
・攻撃１
弾が消えると爆発する 弾は2つしかないので頑張って
・攻撃2
自分中心に回復するサークルを出す
対戦向けに作った

*槍兵
・攻撃１
消滅までの時間は長め 1発しかない
・攻撃2
近接の範囲攻撃だが1とは逆に装弾多めかつリロード早し
なのでメインウェポンはこっちになる

*重火器兵
・攻撃１
装弾数多いけど連射かつすごいぶれるのですぐなくなる
・攻撃2
足が速くなる
対戦実装したらキャラごとの足の速さも変えるつもりなのでその時用

*獣人兵
・攻撃１
弾速早い、よく飛ぶ、リロード長い以上
・攻撃2
自身に攻撃バフとHP回復
回復量多いので実質無敵

*狙撃兵
・攻撃１
クソエイムだと死ぬ
・攻撃2
手榴弾。壁とかに当たっても消えずに跳ね返る
でも、敵の近くで爆発とかいう機能はついてないからこれもエイム大事
